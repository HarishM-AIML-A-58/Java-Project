Make sure your PC has JDK installed before setting up NetBeans:
https://www.oracle.com/java/technologies/downloads/#jdk23-windows

Apache NetBeans: Download it from this link:
https://dlcdn.apache.org/netbeans/netbeans-installers/23/Apache-NetBeans-23-bin-windows-x64.exe

XAMPP: Download from this link:
https://www.apachefriends.org/download.html

MySQL JDBC Driver:
Download the MySQL JDBC driver from this link:
https://dbschema.com/jdbc-driver/mysql.html
After downloading, add it to your project by right-clicking on "Libraries" under your project file, then selecting Add JAR/Folder and importing the driver.

Now your tech stack is ready!
